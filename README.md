# Path of Pain

the best game ever

## Requirements

python 3.5+

pygame 1.9.4+

gamepad (preferable)

## Controls

|   | move  |  attack |  dash | backdash  | exit  |  restart | switch mode  |
| :------------: | :------------: | :------------: | :------------: | :------------: | :------------: | :------------: | :------------: |
| keyboard  | arrows  | s  |  d |  a |  Esc | Tab  | p  |
|  gamepad | LS  | X  | A  | B  | Back  | Start  |  Home |
