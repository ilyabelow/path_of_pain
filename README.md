# Path of Pain

Simple top view slasher build on pygame

### Controls

|    | move  | look around |  attack |  dash | backdash  | interact | exit/menu  |  restart |
| :------------: | :------------: |:------------: | :------------: | :------------: | :------------: | :------------: | :------------: | :------------: |
| keyboard  | arrows  | None |s  |  d |  a | f | Esc | Tab  | 
|  gamepad | LS  | RS  |X  | A  | B | Y | Back  | Start  | 

Works best with gamepad

### Launching the game

> python3 launch.py
